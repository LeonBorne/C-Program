Find Greatest Number among two Numbers

#include<stdio.h>
void main(){
    int a, b, c;
    printf("Enter any three numbers\n");
    scanf("%d %d %d", &a, &b, &c);
    if(a > b && a > c){
        printf("%d is the greatest number.", a);
    }
    else if (b > a && b > c){
        printf("%d is the greatest number.", b);
    }
    else{
        printf("%d is the greatest number.", c);
    }
}

Explanation:

The following C program determines the greatest of three numbers entered by the user. First, the variables a, b, and c are declared as integers to store the input values. The printf() function prompts the user to enter three numbers, and scanf() reads the values into a, b, and c. The program then uses an if-else if-else structure to compare the numbers:

  • If a > b and a > c, it prints that a is the greatest number.
  • Else if b > a and b > c, it prints that b is the greatest number.
  • Otherwise, it prints that c is the greatest number.
The result is displayed using the printf() function.

Improvements:

Currently, the program does not explicitly handle the case when two or more numbers are equal. To handle all cases, you can modify the conditions or use nested if statements.
