C Program to check Positive/Negative number 

#include<stdio.h>
void main(){
    int n;
    printf("Enter a number : ");
    scanf("%d", &n);
    if(n > 0){
        printf("%d is a positive number.", n);
    }
    else if(n < 0){
        printf("%d is a negative number.", n);
    }
    else{
        printf("%d is 0.", n);
    }
}

Explanation:

The following C program determines whether a number entered by the user is positive, negative, or zero. First, the variable n is declared as an integer to store the user input. The printf() function prompts the user to enter a number, and scanf() reads the input into n. The program then uses an if-else if-else structure to check the value of n:

  • If n > 0, it prints that the number is positive.
  • If n < 0, it prints that the number is negative.
  • If none of the above is true, it prints that the number is zero.
The result is displayed using the printf() function.
