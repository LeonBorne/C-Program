Check Alphabet, Special Character or Number

#include<stdio.h>
void main(){
    char a;
    printf("Enter a character : ");
    scanf("%c", &a);
    if((a >= 'a' && a <= 'z') || (a >= 'A' && a <= 'Z')){
        printf("%c is an alphabet.", a);
    }
    else if(a >= '0' && a <= '9'){
        printf("%c is a number.", a);
    }
    else{
        printf("%c is a special character.", a);
    }
}

Explanation:

The following C program determines whether a character entered by the user is an alphabet, a number, or a special character. First, the variable a is declared as char to store the input character. The printf() function prompts the user to enter a character, and scanf() reads the input into a. The program then uses an if-else if-else structure to check the type of the character:

  • If a is between 'a' and 'z' or between 'A' and 'Z', it prints that the character is an alphabet.
  • If a is between '0' and '9', it prints that the character is a number.
  • Otherwise, it prints that the character is a special character.
The result is displayed using the printf() function.
