C Program to Find Volume of Cone

#include<stdio.h>
void main(){
    float pi = 3.14, radius, height, volume;
    printf("Enter the radius of the cone : ");
    scanf("%f", &radius);
    printf("Enter the height of the cone : ");
    scanf("%f", &height);
    volume = (1.0/3.0) * pi * radius * radius * height;
    printf("Volume of the cone : %f", volume);
}

Explanation:

The following C program calculates the volume of a cone based on the radius and height entered by the user. First, the variable pi is initialized with the value 3.14, and the variables radius, height, and volume are declared as float. The printf() and scanf() functions are used to take user input for the radius and height. Next, the expression volume = (1/3) * pi * radius * radius * height computes the volume of the cone. Finally, the result is displayed using the printf() function.

Improvements:

In the formula, (1/3) performs integer division, which always results in 0. To avoid this error, use (1.0/3) or 0.333 so the division is done in floating-point(decimal). Also, to limit the result to two decimal places, you can write:
printf("Volume of the cone : %.2f", volume);
