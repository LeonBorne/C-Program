Check Uppercase or Lowercase Alphabet

#include<stdio.h>
void main(){
    char a;
    printf("Enter an alphabet : ");
    scanf("%c", &a);
    if(a >= 'A' && a <= 'Z'){
        printf("%c is an Uppercase Alphabet.", a);
    }
    else{
        printf("%c is Lowercase ALphabet.", a);
    }
}

Explanation:

The following C program determines whether an alphabet entered by the user is uppercase or lowercase. First, the variable a is declared as char to store the input character. The printf() function prompts the user to enter an alphabet, and scanf() reads the input into a. The program then uses an if-else statement to check the case of the alphabet:

  • If a is between 'A' and 'Z', it prints that the alphabet is uppercase.
  • Otherwise, it prints that the alphabet is lowercase.
The result is displayed using the printf() function.
