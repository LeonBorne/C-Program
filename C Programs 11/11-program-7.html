Find Greatest Number among two Numbers

#include<stdio.h>
void main(){
    int a, b;
    printf("Enter any two numbers\n");
    scanf("%d %d", &a, &b);
    if(a > b){
        printf("%d is greater than %d", a, b);
    }
    else{
        printf("%d is greater than %d", b, a);
    }
}

Explanation:

The following C program determines which of the two numbers entered by the user is greater. First, the variables a and b are declared as integers to store the user input. The printf() function prompts the user to enter two numbers, and scanf() reads the values into a and b. The program then uses an if-else statement to compare the two numbers:

  • If a > b, it prints that a is greater than b.
  • Otherwise, it prints that b is greater than a.
The result is displayed using the printf() function.
