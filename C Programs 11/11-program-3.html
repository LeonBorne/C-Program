C Program to Find Area of Circle

#include<stdio.h>
void main(){
    float pi = 3.14, radius, area;
    printf("Enter the radius of circle : ");
    scanf("%f", &radius);
    area = pi * radius * radius;
    printf("Area of the Circle : %f", area);
}

Explanation:

The following C program calculates the area of a circle based on the radius entered by the user. First, the variable pi is initialized with the value 3.14, and the variables radius and area are declared as float. The printf() function prompts the user to enter the radius of the circle, and the scanf() function stores the input in radius. Next, the expression area = pi * radius * radius computes the area of the circle. Finally, the result is displayed using the printf() function.

Improvements:

By default, printf("Area of the Circle : %f", area) displays the value with many decimal places. To show the result with only two decimal places, you can write:
printf("Area of the Circle : %.2f", area);
