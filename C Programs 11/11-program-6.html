C Program to check Odd/Even Number

#include<stdio.h>
void main(){
    int n;
    printf("Enter a number : ");
    scanf("%d", &n);
    if(n%2 == 0){
        printf("%d is an Even Number.", n);
    }

    else{
        printf("%d is an Odd Number.", n);
    }
}

Explanation:

The following C program determines whether a number entered by the user is even or odd. First, the variable n is declared as an integer to store the user input. The printf() function prompts the user to enter a number, and scanf() reads the input into n. The program then uses an if-else statement to check the value of n:

  • If n % 2 == 0, the number is even, and it prints that n is an even number.
  • Otherwise, the number is odd, and it prints that n is an odd number.
The result is displayed using the printf() function.
