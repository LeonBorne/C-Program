C Program Find Area of Rectangle

#include<stdio.h>
void main(){
    float l, b, area;
    printf("Enter the value of length(l) : ");
    scanf("%f", &l);
    printf("Enter the value of breadth(b) : ");
    scanf("%f", &b);
    area = l * b;
    printf("Area of the rectangle : %f", area);
}

Explanation:

The following C program calculates the area of a rectangle based on user input. First, the variables l, b, and area are declared as float to allow decimal values. The printf() function is used to prompt the user to enter the values of length (l) and breadth (b). The scanf() function then reads these values and stores them in the respective variables. Next, the expression area = l * b computes the area of the rectangle. Finally, the result is displayed using the printf() function.

Improvements:

By default, printf("Area of the rectangle : %f", area) displays the value with many decimal places. To limit the output to two decimal places, you can use:
printf("Area of the rectangle : %.2f", area);
