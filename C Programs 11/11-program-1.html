C Program to Add Two Numbers

#include<stdio.h>
void main(){
    int a, b, sum;
    printf("Enter the value of a : ");
    scanf("%d", &a);
    printf("Enter the value of b : ");
    scanf("%d", &b);
    sum = a + b;
    printf("%d + %d = %d", a, b, sum);
}

Explanation:

The following C program takes two integers as input from the user and displays their sum. First, the variables a, b, and sum are declared as integers. The scanf() function is then used to read the input values into a and b. Next, the expression sum = a + b calculates the sum of the two numbers. Finally, the result is displayed using the printf() function.
