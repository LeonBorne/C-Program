Input details of Student and display it using Structure

#include<stdio.h>

struct student{
    int id;
    char name[20];
    char address[20];
    float marks;
} s;

void main(){
    printf("Input the details of a student\n");
    printf("Enter Id : ");
    scanf("%d", &s.id);
    printf("Enter Name : ");
    scanf("%s", s.name);
    printf("Enter Address : ");
    scanf("%s", s.address);
    printf("Enter marks : ");
    scanf("%f", &s.marks);

    printf("Displaying the detials of the student\n");
    printf("Id : %d\n", s.id);
    printf("Name : %s\n", s.name);
    printf("Address : %s\n", s.address);
    printf("Marks : %.2f\n", s.marks);
}

Explanation:

The following C program demonstrates the use of a struct in C to store and display the details of a student. A structure named student is defined with four members:

  • id (integer) - to store the student's ID
  • name[20] (character array) - to store the student's name
  • address[20] (character array) - to store the student's address
  • marks (float) - to store the student's marks
An object s of type struct student is declared globally. In main(), the program prompts the user to input all the details of the student using scanf(). Finally, the entered details are displayed using printf(). The marks are displayed with %.2f formatting so that only two decimal places are shown.
