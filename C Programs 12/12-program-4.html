Store and Display details of 5 students using structure

#include<stdio.h>

struct student{
    int id;
    char name[20];
    char address[20];
    float marks[3];
} s[5];

void main(){
    int i, j;
    
    printf("Input the details of the students\n");
    for(i = 0; i < 5; i++){
        printf("For Student no. %d\n", i+1);
        printf("Enter Id : ");
        scanf("%d", &s[i].id);
        printf("Enter Name : ");
        scanf("%s", s[i].name);
        printf("Enter Address : ");
        scanf("%s", s[i].address);

        for(j = 0; j < 3; j++){
            printf("Enter marks of subject %d : ", j+1);
            scanf("%f", &s[i].marks[j]);
        }
    }

    printf("Displaying the detials of the student\n");
    for(i = 0; i < 5; i++){
        printf("Details of Student no %d\n", i+1);
        printf("Id : %d\n", s[i].id);
        printf("Name : %s\n", s[i].name);
        printf("Address : %s\n", s[i].address);

        for(j = 0; j < 3; j++){
            printf("Marks of subject %d : %.2f\n", j+1,s[i].marks[j]);
        }
    }
}

Explanation:

The following C program demonstrates the use of struct with arrays to store and display the details of multiple students. A structure named student is defined with the following members:

  • id (integer) - stores the student's ID
  • name[20] (character array) - stores the student's name
  • address[20] (character array) - stores the student's address
  • marks[3] (float array) - stores marks of 3 subjects
An array s[5] is declared to store the details of 5 students. Using a for loop, the program asks the user to input the details of each student. Inside the loop, another for loop is used to input marks for 3 subjects. After input is complete, another set of for loops displays the details of all 5 students, including their marks. The marks are printed using %.2f to display two decimal places.
