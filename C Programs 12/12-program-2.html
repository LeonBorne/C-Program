Check Odd/Even Numbers using Functions

Function without Arguments and without Return type

#include<stdio.h>
void check();
void main(){
    check();
}
void check(){
    int n;
    printf("Enter a number : ");
    scanf("%d", &n);
    if(n%2 != 0){
        printf("%d is an Odd number.", n);
    }
    else{
        printf("%d is an Even number.", n);
    }
}

Explanation:

The following C program demonstrates the use of a function to check whether a number is even or odd. First, the function check() is declared and later defined to perform the check. In main(), the check() function is called to execute its task. Inside the check() function, the variable n is declared as an integer to store the user input. The program prompts the user to enter a number using printf() and scanf(). An if-else statement is used to check whether the number is odd or even:

  • If n % 2 != 0, it prints that the number is odd.
  • Otherwise, it prints that the number is even.
The result is displayed using the printf() function.

Function with Arguments and without Return type

#include<stdio.h>
void check(int n);
void main(){
    int a; 
    printf("Enter a number : ");
    scanf("%d", &a);
    check(a);
}
void check(int n){
    if(n%2 != 0){
        printf("%d is an Odd number.", n);
    }
    else{
        printf("%d is an Even number.", n);
    }
}

Explanation:

The following C program demonstrates the use of a function with a parameter to check whether a number is even or odd. First, the function check(int n) is declared to accept an integer as input. In main(), the variable a is declared as an integer to store user input. The program prompts the user to enter a number using printf() and scanf(). The check(a) function is then called, passing the input value as an argument. Inside the check() function, an if-else statement checks whether the number is odd or even:

  • If n % 2 != 0, it prints that the number is odd.
  • Otherwise, it prints that the number is even.
The result is displayed using the printf() function.

Function without Arguments and with Return type

#include<stdio.h>
int check();
void main(){
    int result = check();
    if(result == 1){
        printf("The number is an Odd number.");
    }
    else{
        printf("The number is an Even Number.");
    }
}
int check(){
    int n;
    printf("Enter a number : ");
    scanf("%d", &n);
    if(n%2 != 0){
        return 1;
    }
    else{
        return 0;
    }
}

Explanation:

The following C program demonstrates the use of a function that returns a value to determine whether a number is even or odd. First, the function check() is declared to return an integer. In main(), the return value of check() is stored in the variable result. Inside the check() function, the variable n is declared as an integer. The program prompts the user to enter a number using printf() and scanf(). An if-else statement checks whether the number is odd or even:

  • If n % 2 != 0, the function returns 1.
  • Otherwise, the function returns 0.
Back in main(), result is used in an if-else statement to print whether the number is odd or even.

Function with Arguments and with Return type

#include<stdio.h>
int check(int n);
void main(){
    int a;
    printf("Enter a number : ");
    scanf("%d", &a);
    int result = check(a);
    if(result == 1){
        printf("The number is an Odd number.");
    }
    else{
        printf("The number is an Even Number.");
    }
}
int check(int n){
    if(n%2 != 0){
        return 1;
    }
    else{
        return 0;
    }
}

Explanation:

The following C program demonstrates the use of a parameterized function that returns a value to determine whether a number is even or odd. First, the function check(int n) is declared to accept an integer and return 1 if it is odd or 0 if it is even. In main(), the variable a is declared to store user input. The program prompts the user to enter a number using printf() and scanf(). The check(a) function is called with the input value a, and its return value is stored in result. An if-else statement in main() uses result to print whether the number is odd or even.
