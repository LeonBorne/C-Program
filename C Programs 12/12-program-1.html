C Program to Add Two Numbers using Functions

Function without Argument and without Return type

#include<stdio.h>
void add();
void main(){
    add();
}
void add(){
    int a, b, sum;
    printf("Enter the value of a : ");
    scanf("%d", &a);
    printf("Enter the value of b : ");
    scanf("%d", &b);
    sum = a + b;
    printf("The sum of two numbers is : %d", sum);
}

Explanation:

The following C program demonstrates the use of a function to add two numbers. First, a function add() is declared and later defined to perform the addition. In main(), the function add() is called to execute its task. Inside the add() function, the variables a, b, and sum are declared as integers. The program prompts the user to enter the values of a and b using printf() and scanf(). Finally, it prints the sum of a and b using printf().

Function with Arguments and without return type

#include<stdio.h>
void add(int a, int b);
void main(){
    int x, y;
    printf("Enter two numbers\n");
    scanf("%d %d", &x, &y);
    add(x, y);
}
void add(int a, int b){
    int sum = a + b;
    printf("The Sum of two numbers is : %d", sum);
}

Explanation:

The following C program demonstrates the use of a function with parameters to add two numbers. First, the function add(int a, int b) is declared to accept two integer arguments. In main(), the variables x and y are declared to store user input. The program prompts the user to enter two numbers using printf() and scanf(), and then calls the add(x, y) function, passing the input values as arguments. Inside the add() function, the sum of the two numbers is calculated and displayed using printf().

Function without Arguments and with return type

#include<stdio.h>
int add();
void main(){
    int sum = add();
    printf("The Sum of two numbers is : %d", sum);
}
int add(){
    int a, b;
    printf("Enter two numbers\n");
    scanf("%d %d", &a, &b);
    return (a + b);
}

Explanation:

The following C program demonstrates the use of a function that returns a value. First, the function add() is declared to return an integer. In main(), the function add() is called and its return value is stored in the variable sum. Inside the add() function, the variables a and b are declared as integers. The program prompts the user to enter two numbers using printf() and scanf(). The function then returns the sum of the two numbers using return (a + b);, which is stored in sum in main(). Finally, the sum is displayed using printf().

Function With Arguments and Return type

#include<stdio.h>
int add(int a, int b);
void main(){
    int x, y, sum;
    printf("Enter two numbers\n");
    scanf("%d %d", &x, &y);
    sum = add(x, y);
    printf("The sum of two numbers is : %d", sum);
}
int add(int a, int b){
    return (a + b);
}

Explanation:

The following C program demonstrates the use of a function with parameters that returns a value. First, the function add(int a, int b) is declared to accept two integers and return their sum. In main(), the variables x, y, and sum are declared. The program prompts the user to enter two numbers using printf() and scanf(). The add(x, y) function is then called with x and y as arguments, and its return value is stored in sum. Finally, the sum is displayed using printf().
